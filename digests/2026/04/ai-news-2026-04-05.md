# AI News Digest - 2026-04-05

## Claude Code / Anthropic

### Claude Code v2.1.90〜v2.1.92 リリース（4/1〜4/4）
前回チェック（v2.1.89）以降、3バージョンがリリースされた。主な変更点:

- **v2.1.92（4/4）**: Bedrock対話型セットアップウィザード追加、`/cost`でモデル別・キャッシュヒット内訳表示、`forceRemoteSettingsRefresh`ポリシー設定追加（起動時にリモート設定を強制取得、失敗時は終了）、Writeツールの差分計算が60%高速化
- **v2.1.91（4/2）**: MCP toolの結果サイズ上限を`_meta["anthropic/maxResultSizeChars"]`で最大500Kまでオーバーライド可能に、`disableSkillShellExecution`設定追加、Editツールがより短い`old_string`アンカーを使用しトークン削減
- **v2.1.90（4/1）**: `/powerup`コマンド追加（機能紹介のインタラクティブレッスン）、`--resume`時のプロンプトキャッシュミス修正（v2.1.69以降のリグレッション）、SSEトランスポートの大フレーム処理がO(n²)→O(n)に改善、auto modeがユーザー境界指示（"pushしないで"等）を無視する問題を修正

- https://code.claude.com/docs/en/changelog

### Anthropic、サードパーティツールへのサブスクリプション適用を終了（4/4）
Claude Code等のサブスクリプションが、OpenClawなどサードパーティハーネスでの利用をカバーしなくなった。4/4 12pm PT発効。今後はAPI利用または従量課金（extra usage）が必要。月額サブスクリプション相当の一時クレジット提供あり（4/17まで）。Claude Code責任者Boris Chernyは「サブスクリプションはサードパーティツールの利用パターンを想定していなかった」と説明。

- https://techcrunch.com/2026/04/04/anthropic-says-claude-code-subscribers-will-need-to-pay-extra-for-openclaw-support/

## コーディングエージェント

### Cursor 3（コードネーム: Glass）ローンチ（4/2）
Cursorがバージョン3をリリース。UIを一新し、開発者がAIエージェントをオーケストレーションする前提で再設計。マルチリポジトリレイアウト、並列エージェント実行、ローカル↔クラウド間のハンドオフ（Cloud Handoff）機能を搭載。Design Modeも追加。Claude Code（市場シェア約54%）やOpenAI Codexへの対抗策。料金はPro $20/月、Pro+ $60、Ultra $200。

- https://startupnews.fyi/2026/04/02/cursor-launches-a-new-ai-agent-experience-to-take-on-claude-code-and-codex/

### Claw Code — OSSコーディングエージェントフレームワーク公開（4/2）
Claude Codeのソースマップ流出（3/31）を契機に、クリーンルーム実装によるOSSエージェントハーネスが公開。Python + Rustで構築。公開数日で72,000 GitHub Stars。Claude Codeと同等のアーキテクチャを完全オープンに提供する狙い。Anthropicとは無関係。

- https://claw-code.codes/

### Devin アップデート（4/1, 4/3）
- **4/3**: エンタープライズスコープのシークレット管理、ACU可視性制御、MCPサーバー許可リスト強制、ビルドのピン留め/ロールバック、PRレビュー自動修正機能
- **4/1**: 既存PR引き継ぎ（別セッションで作成されたPRの作業を継続可能）、ターミナル出力のリアルタイムストリーミング

- https://docs.devin.ai/release-notes/overview

## オープンモデル / インフラ

### Google Gemma 4 リリース（4/2）
Googleが最も高性能なオープンモデルファミリー「Gemma 4」を公開。4モデル構成: 2B/4B（Effective、エッジデバイス向け）、26B（MoE）、31B（Dense）。256Kコンテキスト、ネイティブ視覚・音声処理、140言語以上対応。Apache 2.0ライセンス。Android Gemini Nanoの次世代基盤にもなる。

- https://blog.google/innovation-and-ai/technology/developers-tools/gemma-4/

### Microsoft MAIモデル3種リリース（4/3）
Microsoft自社開発の基盤モデル3種を公開。MAI-Transcribe-1（音声→テキスト、FLEURSベンチマークで最低WER、Whisper-large-v3を全25言語で上回る）、MAI-Voice-1（テキスト→音声、1秒で60秒の音声生成、$22/100万文字）、MAI-Image-2（画像生成、2倍高速化）。Microsoft FoundryとMAI Playgroundで利用可能。

- https://venturebeat.com/technology/microsoft-launches-3-new-ai-models-in-direct-shot-at-openai-and-google
