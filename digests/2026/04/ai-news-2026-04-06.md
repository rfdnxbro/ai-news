# AI News Digest: 2026-04-06

## Anthropic / Claude

### Anthropic、サードパーティツールへのサブスクリプション適用を遮断
- 4月4日正午（PT）より、Claude サブスクリプションを OpenClaw 等のサードパーティツールで利用できなくなった
- 理由: サードパーティツールがプロンプトキャッシュを活用できず、Anthropic の計算リソースを圧迫していたため
- 今後は従量課金（pay-as-you-go）での利用が必要。影響ユーザーには全額返金＋1ヶ月分クレジットを提供
- Claude Code 責任者 Boris Cherny が公式に説明
- https://venturebeat.com/technology/anthropic-cuts-off-the-ability-to-use-claude-subscriptions-with-openclaw-and
- https://techcrunch.com/2026/04/04/anthropic-says-claude-code-subscribers-will-need-to-pay-extra-for-openclaw-support/

### Claude Code v2.1.92 リリース（4月4日）
- `forceRemoteSettingsRefresh` ポリシー設定を追加（リモート管理設定のfail-closed取得）
- Bedrock セットアップウィザード追加（AWS認証・リージョン設定のインタラクティブ構成）
- `/cost` でモデル別・キャッシュヒット内訳を表示（サブスクリプションユーザー向け）
- Write ツールの diff 計算が60%高速化
- `/tag` と `/vim` コマンドを削除
- Linux サンドボックスに `apply-seccomp` ヘルパー同梱
- 多数のバグ修正（サブエージェント生成、ストリーミング中のツール入力検証、iTerm2/Ghostty のメッセージ重複等）
- https://code.claude.com/docs/en/changelog

## Cursor

### Cursor 3.0 リリース（4月2日）— UIを全面刷新
- **Agents Window**: 複数エージェントをローカル・クラウド・worktree・リモートSSH環境で並列実行可能
- **Design Mode**: ブラウザ上のUI要素を直接アノテーション・ターゲティングし、エージェントに正確なフィードバックを提供
- **Agent Tabs**: エディタ内でチャットを並列表示（サイドバイサイド/グリッド）
- `/worktree` コマンド（git worktree 分離）、`/best-of-n` コマンド（モデル並列比較）を追加
- 価格据え置き（Pro $20/月）
- https://cursor.com/changelog/3-0
- https://cursor.com/blog/cursor-3
