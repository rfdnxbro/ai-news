# AI News Digest - 2026-04-01

## Claude Code / Anthropic

### Claude Code ソースコード流出（npm sourcemap経由）
Anthropicの`@anthropic-ai/claude-code` v2.1.88のnpmパッケージに、デバッグ用の59.8MBソースマップファイルが誤って同梱され、約512,000行のTypeScriptソースコードが公開状態になった。セキュリティ研究者Chaofan Shou氏がX上で発見・公開し、数時間でGitHubにミラーが拡散。44個の未公開フィーチャーフラグが含まれ、以下の未発表機能が判明：
- **KAIROS**: 常駐デーモンモード。バックグラウンドで動作し、ユーザーがアイドル時に「autoDream」でメモリ統合を行う自律型エージェント
- **Buddy**: たまごっち風コンパニオンペットシステム。18種のキャラクター、レアリティ、シャイニー、ステータス値あり
- **ULTRAPLAN**: 30分リモートプランニング、コーディネーターモード、エージェントスウォーム等
- テスター窓: 4/1〜4/7、本格ローンチは5月（Anthropic社員から開始予定）
- これはAnthropicにとって1週間で2度目の情報流出（先日のモデルスペック流出に続く）

**注意**: 3/31 00:21〜03:29 UTC間にnpm経由でClaude Codeを更新した場合、悪意あるaxiosパッケージ（RAT含む）が混入した可能性あり。該当する場合はシークレットのローテーションが必要。
