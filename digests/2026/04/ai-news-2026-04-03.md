# AI News Digest - 2026-04-03

## Cursor 3.0 リリース — エージェント特化の新インターフェース

Cursor が大型アップデート「Cursor 3.0」を4月2日にリリース。Claude Code や OpenAI Codex に対抗するエージェント体験を前面に押し出した。主な新機能:

- **Agents Window**: 複数のAIエージェントをリポジトリ・環境横断で並列実行
- **Design Mode**: ブラウザ上でUI要素を直接アノテーション・ターゲット指定
- **Agent Tabs**: 複数チャットを並列・グリッド表示
- **`/worktree`・`/best-of-n` コマンド**: ワークツリーでの分離タスク実行
- **MCP Apps**: 構造化コンテンツ対応でリッチなツール出力が可能に
- **Enterprise Admin**: チームシークレット管理・コード帰属設定

https://cursor.com/changelog

## Claude Code v2.1.90 リリース

4月1日リリース。新機能とパフォーマンス改善が中心。

- **`/powerup`**: Claude Code の機能をアニメーション付きで学べるインタラクティブレッスン
- **パフォーマンス改善**: MCP ツールスキーマの JSON.stringify 排除、SSE 大フレーム処理が線形時間に改善（従来は二次時間）、長い会話のトランスクリプト書き込み高速化、`/resume` のセッション並列読み込み
- **主要バグ修正**: レート制限ダイアログの無限ループ、`--resume` 時のプロンプトキャッシュミス、format-on-save フック後のEdit/Write失敗、auto mode がユーザー指示の境界を無視する問題
- **セキュリティ**: PowerShell ツール権限チェック強化、DNS キャッシュ関連コマンドの自動許可を削除

https://code.claude.com/docs/en/changelog
