# AI News Digest - 2026-04-04

## コーディングエージェント・開発ツール

### Cursor 3.0 リリース（4/2）
複数エージェントの並列実行を中心に据えた完全リニューアル。ローカル、git worktree、クラウド、リモートSSHの各環境で同時にエージェントを実行できる「Agents Window」が目玉。ブラウザ上のUI要素を直接指定できる「Design Mode」、エディタ内で複数チャットを並列表示する「Agent Tabs」も追加。`/worktree`（isolatedなgit worktree作成）と`/best-of-n`（複数モデルで同一タスクを比較実行）コマンドが新設。エンタープライズ向けには監査ログ改善、シークレット管理制限、コード帰属表示の無効化が追加。
- https://cursor.com/changelog

### Claude Code v2.1.91（4/2）
MCP toolの結果サイズ上限を`_meta["anthropic/maxResultSizeChars"]`で最大500Kまで拡張可能に。DBスキーマ等の大きな結果がトランケーションされなくなる。スキル内シェル実行を無効化する`disableSkillShellExecution`設定、プラグインの`bin/`配下の実行ファイルをBashツールから直接呼び出せる機能も追加。Editツールの`old_string`アンカーが短縮され出力トークン削減。
- https://code.claude.com/docs/en/changelog

### Claude Code v2.1.90（4/1）
`/powerup`コマンド追加 — Claude Codeの機能をアニメーションデモ付きで学べるインタラクティブレッスン。レートリミットダイアログの無限ループ修正、`--resume`時のプロンプトキャッシュミス修正（v2.1.69からのリグレッション）、PostToolUseフォーマットフック後のEdit/Write失敗修正など多数のバグ修正。PowerShellのツールパーミッションチェック強化。SSEトランスポートの大規模フレーム処理がO(n²)→O(n)に改善。
- https://code.claude.com/docs/en/changelog

### Claw Code — OSSコーディングエージェントフレームワーク公開（4/2）
Claude Codeのソースマップ流出（3月）を受けて、クリーンルーム方式でエージェントハーネスアーキテクチャを再実装したOSSプロジェクト。Python + Rust製で、複数LLMプロバイダ（Claude、OpenAI、ローカルモデル）をサポート。公開数日でGitHub 72,000スター超。独立監査によりAnthropicのプロプライエタリコード未使用を確認済み。
- https://claw-code.codes/

### Devin — PR再開機能ほか改善（4/1）
既存のPRに対してセッションをまたいで作業を継続できる「PR Resuming」機能を追加。「lines left to review」カウンターによるPRレビュー高速化、リアルタイムターミナルストリーミング、リデザインされた設定ページなども含む。
- https://docs.devin.ai/release-notes/overview

## AI基盤・API

### OpenAI Responses API — エージェント機能拡張
Responses APIにシェルツール（Debian 12コンテナ、Python/Node.js/Java等プリインストール）、ビルトインエージェント実行ループ、サーバーサイドコンテキスト圧縮（compaction）、再利用可能なエージェントスキルを追加。compactionにより500万トークン・150ツールコールのセッションでも精度低下なしで実行可能に。Early Bird終了は4/14。
- https://openai.com/index/equip-responses-api-computer-environment/
