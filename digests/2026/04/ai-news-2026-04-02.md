# AI News Digest - 2026-04-02

## Anthropic / Claude

### Claude Code ソースコード流出事件
- 3/31にリリースされたv2.1.88のnpmパッケージに、59.8MBのソースマップファイルが誤って同梱され、512,000行超・2,000ファイルのTypeScriptソースが公開された
- 原因はAnthropicが2025年末に買収したBunのビルドツールの既知バグ（本番ビルドでもソースマップが含まれる問題が20日間未修正だった）
- Anthropicは「リリースパッケージングの問題であり、顧客データや認証情報の漏洩はない」と声明
- コード内から「Proactive mode」フラグ、暗号通貨ベースのエージェント決済システム、`ANTI_DISTILLATION_CC`フラグなどの未発表機能の痕跡が発見された
- GitHubへの著作権テイクダウン要請が想定以上のリポジトリに影響し、後に大幅に縮小された
- 出典: [Bloomberg](https://www.bloomberg.com/news/articles/2026-04-01/anthropic-accidentally-releases-source-code-for-claude-ai-agent), [The Hacker News](https://thehackernews.com/2026/04/claude-code-tleaked-via-npm-packaging.html), [BleepingComputer](https://www.bleepingcomputer.com/news/artificial-intelligence/claude-code-source-code-accidentally-leaked-in-npm-package/)

### Claude Code v2.1.89（4/1リリース）
- `PreToolUse`フックに`"defer"`パーミッション決定を追加。ヘッドレスセッションでツール呼び出し時に一時停止し、`-p --resume`で再評価が可能に
- `CLAUDE_CODE_NO_FLICKER=1`環境変数でフリッカーフリーのalt-screenレンダリングを提供
- `PermissionDenied`フックを追加（autoモード分類器の拒否後に発火、`{retry: true}`で再試行可能）
- 名前付きサブエージェントが`@`メンションのtypeahead候補に追加
- `MCP_CONNECTION_NONBLOCKING=true`で`-p`モード時のMCP接続待機をスキップ可能（タイムアウト5秒）
- 多数のバグ修正: Windows CRLF二重化、StructuredOutputスキーマキャッシュの約50%失敗率、大規模JSON入力時のメモリリーク、CJK/絵文字のプロンプト履歴切り捨て等
- `/buddy` エイプリルフールイースターエッグ（コーディング中に反応する小さな生き物を孵化）
- 出典: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

## OpenAI

### Responses APIのエージェント機能拡張
- シェルツール、ビルトインエージェント実行ループ、ホステッドコンテナワークスペース、コンテキスト圧縮、再利用可能なエージェントスキルを追加
- `container_auto`オプションでDebian 12環境（Python 3.11, Node.js 22, Java 17, Go 1.23, Ruby 3.1）を自動プロビジョニング
- サーバーサイドコンパクションにより、Triple Whaleのエージェント「Moby」が500万トークン・150ツール呼び出しのセッションを精度低下なしに処理
- OpenAI/Anthropic両社がスキル標準（SKILL.mdマニフェスト+YAMLフロントマター）で収斂
- 出典: [OpenAI](https://openai.com/index/new-tools-for-building-agents/), [VentureBeat](https://venturebeat.com/orchestration/openai-upgrades-its-responses-api-to-support-agent-skills-and-a-complete), [InfoQ](https://www.infoq.com/news/2026/03/openai-responses-api-agents/)
