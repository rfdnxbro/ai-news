# AI News Digest - 2026-03-23

## OpenAI — Superapp構想 & Astral買収

### デスクトップSuperapp計画（3/19-20報道）
- ChatGPT・Codex・Atlasブラウザを単一のデスクトップアプリに統合する計画をWSJが報道
- アプリケーション部門CEO Fidji Simo主導。「複数アプリに力を分散しすぎた」と社内メモ
- 背景: 初めてAIツールを導入する企業の支出でAnthropicが73%を獲得、OpenAIは約27%に低下（Axios調査）
- まずCodexにコーディング以外のエージェント機能を追加し、その後ChatGPT・Atlasを統合。モバイル版は変更なし
- リリース時期は未定
- https://www.cnbc.com/2026/03/19/openai-desktop-super-app-chatgpt-browser-codex.html

### Astral買収（3/19発表）
- Python開発ツールuv（パッケージマネージャ）、Ruff（リンター）、ty（型チェッカー）のAstralを買収。Codex強化が目的
- Astralチームは全員Codexチームに合流。uv/Ruff/tyはMIT/Apache 2.0ライセンスで引き続きOSS維持を表明
- uvは先月のPyPIダウンロード数1.26億回。Codexは週間アクティブユーザー200万人超
- コミュニティからはロードマップの独立性・ガバナンスの不透明さへの懸念あり
- 参考: Anthropicも2025年にJavaScript開発ツール企業Bunを買収済。開発者体験の競争が激化
- https://openai.com/index/openai-to-acquire-astral/

## Claude Code — Channels機能（3/20リリース）

- Telegram・Discordからローカルで動作中のClaude Codeセッションにメッセージを送受信できる新機能。リサーチプレビュー
- ローカルのファイルシステム・MCP・gitアクセスをフル活用しつつ、モバイルから指示可能
- MCP（Model Context Protocol）ベースのプラグインアーキテクチャで実装。ローカルからBotAPIをポーリングする方式（ポート開放不要）
- 要件: Claude Code v2.1.80以降、Bunランタイム、claude.aiアカウント（APIキー不可）
- 制限: ターミナルセッションの維持が必要、権限確認はターミナル側でのみ対応
- OpenClawのモバイルアクセス機能への対抗と位置づけられている
- https://venturebeat.com/orchestration/anthropic-just-shipped-an-openclaw-killer-called-claude-code-channels

## Manus — "My Computer" デスクトップアプリ（3/16-18）

- Meta傘下のManus（旧中国→シンガポール拠点）がデスクトップアプリ「My Computer」をリリース
- AIエージェントがローカルのCLI経由でファイル操作・アプリ制御・Google連携等を自律実行
- Mac/Windowsに対応。実行前に明示的な承認を要求するセーフティコントロール付き
- 無料プランあり、有料は$20/月〜。OpenClawとの差別化はMetaの独自モデルスタックによる一貫性
- https://www.cnbc.com/2026/03/18/metas-manus-launches-desktop-app-to-bring-its-ai-agent-onto-personal-devices.html
