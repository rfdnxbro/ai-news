## Anthropic / Claude

### Claude Mythos（Capybara）モデルのリーク
Anthropicの内部CMS設定ミスにより、開発中の新モデル「Claude Mythos」の情報が流出。Opusの上位に位置する新ティア「Capybara」に属し、コーディング・学術推論・サイバーセキュリティで既存モデルを大幅に上回るとされる。Fortuneが3/26に報道、Anthropicは「ステップチェンジ」と認め、当面はサイバーセキュリティ防御用途の限定提供を予定。運用コストが非常に高く、一般公開前に効率化が必要とのこと。同日、AnthropicがIPOを2026年10月にも検討中との報道も。
- https://fortune.com/2026/03/26/anthropic-says-testing-mythos-powerful-new-ai-model-after-data-leak-reveals-its-existence-step-change-in-capabilities/
- https://fortune.com/2026/03/27/anthropic-leaked-ai-mythos-cybersecurity-risk/

### Claude Code v2.1.86（3/27リリース）
主な変更点:
- APIリクエストに`X-Claude-Code-Session-Id`ヘッダー追加（プロキシでのセッション集約用）
- `.jj`/`.sl`をVCSディレクトリ除外リストに追加（Jujutsu/Sapling対応）
- `--resume`失敗、`--bare`モードのMCPツール欠落、Windows設定破損、`/feedback`のOOMクラッシュ等の修正
- VSCode拡張: 長時間操作中の「Not responding」表示を修正
- Readツールがコンパクトな行番号形式に変更、未変更の再読み込みを重複排除
- メモリ増加（markdown/highlightレンダーキャッシュ）の修正
- https://code.claude.com/docs/en/changelog

### Claude Codeレートリミット問題
3/23以降、Max契約ユーザーから「5時間セッションが1-2時間で枯渇」「1プロンプトで使用率が急増」との報告が多発。Anthropicはピーク時間帯（PT 5am-11am）の意図的な制限強化と説明。背景にはQuitGPT運動による流入増（Webトラフィック30%↑）がある。オフピーク時の使用量倍増プロモーションを3/27まで実施。
- https://www.macrumors.com/2026/03/26/claude-code-users-rapid-rate-limit-drain-bug/
- https://github.com/anthropics/claude-code/issues/38335

## 開発ツール・エージェント

### JetBrains Central発表（3/24）
AIコーディングエージェントの管理・実行プラットフォーム。Junie、Claude Agent、Codex、Gemini CLIなど外部エージェントも統合可能。ガバナンス・ポリシー管理、セマンティックコンテキスト、インテリジェントルーティングを提供。Q2 2026にEAP開始予定。Code With Meは廃止へ（2027 Q1にリレーインフラ終了）。
- https://blog.jetbrains.com/blog/2026/03/24/introducing-jetbrains-central-an-open-system-for-agentic-software-development/

### OpenAI Codex Plugins & Codex Security
Codexにプラグインシステムを導入。Slack・Figma・Notion・Gmail等20+の初期統合。スキル（個人用）とプラグイン（チーム共有・バージョン管理）を明確に分離。Codex Securityは30日間で120万コミットをスキャンし、792件のクリティカル・10,561件の高重要度の脆弱性を発見。
- https://openai.com/index/codex-security-now-in-research-preview/
- https://windowsreport.com/openai-expands-codex-with-shareable-plugins-and-team-workflow-tools/
