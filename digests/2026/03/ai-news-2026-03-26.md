# AI News Digest - 2026-03-26

## Claude Code / Anthropic

### Claude Code v2.1.83 リリース（3/25）
- `managed-settings.d/` ドロップインディレクトリ追加。チームポリシーをファイル分割して管理可能に
- `CwdChanged`・`FileChanged` フックイベント追加でリアクティブな環境管理が可能に
- `sandbox.failIfUnavailable` 設定追加（サンドボックス起動失敗時にエラー終了）
- `CLAUDE_CODE_SUBPROCESS_ENV_SCRUB=1` でサブプロセスから認証情報を除去
- `Ctrl+X Ctrl+E` で外部エディタ起動、画像貼り付け時に `[Image #N]` チップ挿入
- エージェントが `initialPrompt` フロントマターで初回ターン自動送信可能に
- Bedrock コールドスタート改善、`--resume` メモリ使用量改善等のパフォーマンス修正
- Source: https://code.claude.com/docs/en/changelog

### Claude モバイルアプリ: インタラクティブアプリ対応（3/25）
- iOS/Androidアプリで会話内にライブチャート・ダイアグラム・共有可能なアセットを直接描画可能に
- Source: https://support.claude.com/en/articles/12138966-release-notes

### Claude Computer Use リサーチプレビュー（3/23）

- Source: https://support.claude.com/en/articles/12138966-release-notes
