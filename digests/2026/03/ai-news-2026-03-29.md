## Anthropic / Claude

### Claude Code v2.1.83〜v2.1.86 リリース（3/25〜3/27）
前回チェック（v2.1.81）以降、4バージョンがリリース。主な新機能・改善:
- **PowerShell ツール（Windows向けopt-inプレビュー）** — v2.1.84
- **条件付きhooks（`if`フィールド）** — v2.1.85。パーミッションルール構文で発火条件を指定可能
- **`CwdChanged` / `FileChanged` hookイベント** — v2.1.83。リアクティブな環境管理向け
- **トランスクリプト検索（`/`キー）** — v2.1.83
- **`managed-settings.d/` ドロップインディレクトリ** — v2.1.83。ポリシーフラグメント管理
- **MCP OAuth が RFC 9728 Protected Resource Metadata に準拠** — v2.1.85
- **大量のバグ修正・パフォーマンス改善**: メモリリーク修正、スクロール性能向上（WASM→TypeScript）、起動高速化、VSCode拡張の安定性向上など
- ソース: https://code.claude.com/docs/en/changelog
